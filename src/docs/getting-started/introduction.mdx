[AirGap](https://airgap.it) is a crypto wallet system that lets you secure crypto assets with an offline device.

The [AirGap Vault](airgap-vault/introduction) application is installed on a dedicated device that has no connection to any network, thus it is air gapped.

The [AirGap Wallet](airgap-wallet/introduction) is installed on your everyday smartphone.
