If you hold ATOM in AirGap, you can stake them to earn rewards. See the following guide for more info: https://medium.com/airgap-it/secure-cosmos-delegation-62a4b9acdd62

| Action               | Value                                                                |
| -------------------- | -------------------------------------------------------------------- |
| Locked while staking | Yes                                                                  |
| Undelegating period  | 21 days                                                              |
| Rewards after        | a few days                                                           |
| Payout               | Needs to be claimed                                                  |
| Slashing Risk        | Yes (If Validator gets slashed, you lose a percentage of your funds) |
