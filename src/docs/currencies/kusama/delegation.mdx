---
title: Delegation
---

If you hold KSM in AirGap, you can stake them to earn rewards. See the following guide for more info: https://medium.com/airgap-it/airgap-adds-polkadot-support-with-delegation-a4d14a8b55d3

| Action               | Value                                                                                    |
| -------------------- | ---------------------------------------------------------------------------------------- |
| Locked while staking | Yes                                                                                      |
| Undelegating period  | 7 days                                                                                   |
| Rewards after        | 6 hours                                                                                  |
| Slashing Risk        | Yes (If Validator gets slashed, you lose a percentage of your funds)                     |
| Payout               | Needs to be claimed by someone (Usually done by the Validator, so no claiming necessary) |
