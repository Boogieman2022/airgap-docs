## Introduction to Moonbeam

Moonbeam is a para-chain in the Polkadot ecosystem. It is an L1 blockchain solution that extends the base Ethereum featureset with additional features such as on-chain governance, staking, and cross-chain integrations.
GLMR is the official token of the moonbeam ecosystem, and it serves the following purposes:

- Payment of transaction fees: All transaction fees on the Moonbeam ecosystem are paid using GLMR.
- Execution of smart contracts: GLMR is used to pay for smart contract execution, playing the same role that ETH and BNB on Ethereum and BSC.
- The incentive for collators to produce blocks to support the network: Nomination of collators which helps the liveness of the network is done using GLMR token.
- Facilitation of Moonbeamâ€™s on-chain governance system: Participation in on-chain Glimmer-weighted voting and governance is done using GLMR token.
