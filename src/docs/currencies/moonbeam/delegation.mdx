If you hold GLMR in AirGap, you can delegate them to earn rewards following this guide.

## key considerations before staking

| Action               | Value         |
| -------------------- | ------------- |
| Locked while staking | Yes           |
| Undelegating period  | 7 days        |
| Rewards after        | 12 - 18 hours |
| Slashing Risk        | No            |
| Payout               | Automatic     |
| Minimun              | 50 GLMR       |
| Max                  | 300 GLMR      |
| Auto-compound        | No            |
| Commission fee       | 20% of profit |
