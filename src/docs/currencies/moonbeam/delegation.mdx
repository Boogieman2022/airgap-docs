If you hold GLMR in AirGap, you can delegate them to earn rewards following this guide.

## key considerations before staking

| Action               | Value         |
| -------------------- | ------------- |
| Locked while staking | Yes           |
| Undelegating period  | 7 days        |
| Rewards after        | 12 - 18 hours |
| Slashing Risk        | No            |
| Payout               | Automatic     |
| Minimun              | 50 GLMR       |
| Max                  | 300 GLMR      |
| Auto-compound        | No            |
| Commission fee       | 20% of profit |

## How to stake GLMR in AirGap Wallet

- To begin staking GLMR, ensure you have set up a GLMR account. To set up a GLMR account, follow this [guide](https://support.airgap.it/guides/step-by-step-guide). The guide uses XTZ as an example, but the same process can be applied to GLMR.

- Open your wallet and select your GLMR account. 
- Click the delegation button.

:::note Note
The first two images show a balance of 3 GLMR, but there is a required minimum of 50 GLMR.
:::
