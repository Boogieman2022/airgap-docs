title: Delegation
---

If you hold DOT in AirGap, you can delegate them to earn rewards. See the following guide for more info: https://medium.com/airgap-it/airgap-one-transaction-tezos-delegation-dd66f8f338c7

|                      |                |
| -------------------- | -------------- |
| Locked while staking | No             |
| Undelegating period  | No             |
| Rewards after        | Around 21 days |
